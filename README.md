# 3-Column Preview Card Component

This is my solution to the 3-Column Preview Card Component challenge from Frontend Mentor.

## Overview

The project displays three responsive preview cards, each highlighting a different vehicle category:
- Sedans
- SUVs
- Luxury

Each card includes an icon, title, description, and a call-to-action button.

## Built With

- HTML5
- CSS3
- Flexbox

## Features

- Clean and modern card design
- Responsive layout
- Semantic HTML structure
- Custom styling with CSS
- Interactive hover effects
